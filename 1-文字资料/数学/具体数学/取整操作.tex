\textbf{核心互换恒等式}

\[\left\lfloor x \right\rfloor = -\left\lceil -x \right\rceil. \\[0.6em]
\left\lceil x \right\rceil = -\left\lfloor -x \right\rfloor.\]



\textbf{常见分式公式}

\[\left\lceil \frac{a}{b} \right\rceil = \left\lfloor \frac{a + b - 1}{b} \right\rfloor. \\[0.6em]
\left\lfloor \frac{a}{b} \right\rfloor = \left\lceil \frac{a - b + 1}{b} \right\rceil.\]


\textbf{常见不等式}
加法：

\[\left\lfloor x \right\rfloor + \left\lfloor y \right\rfloor 
\le \left\lfloor x + y \right\rfloor 
\le \left\lfloor x \right\rfloor + \left\lfloor y \right\rfloor + 1.\]

整数乘法 \(n\in\mathbb Z_+\)：

\[n\left\lfloor x \right\rfloor 
\le \left\lfloor nx \right\rfloor
\le n\left\lfloor x \right\rfloor + (n - 1).\]

Hermite拆分：

\[\left\lfloor nx \right\rfloor = \sum_{k = 0}^{n - 1} \left\lfloor x + \frac{k}{n} \right\rfloor.\]
